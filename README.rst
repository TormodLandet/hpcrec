Harmonic Polynomial Cell
========================

This Python code implements the HPC method by `Shao and Faltinsen <https://scholar.google.com/scholar?q=Yan-Lin+shao+Odd+faltinsen+harmonic>`_.
The code also contains an interface with FEniCS to benchmark the HPC method's harmonic polynomial interpolation vs the more standard continuous Galerkin FEM method with Lagrange polynomials. 


The HPC method
--------------

The HPC method solves the Laplace equation for an unknown function :math:`\phi` on a domain :math:`\Omega` completely enclosed by non-overlapping boundaries :math:`\Gamma_D` and :math:`\Gamma_N` such that :math:`\partial\Omega = \Gamma_D \cup \Gamma_N`. 
The equations to solve are hence

.. math::

    \nabla^2 \phi &= 0 \quad \text{ in } \Omega \\
    \phi &= g_D \quad \text{ on } \Gamma_D \\
    \frac{\partial \phi}{\partial \mathbf{n}} &= g_N \quad \text{ on } \Gamma_N.
    
We approximate the domain :math:`\Omega` by a tesselation :math:`\mathcal{T}` which consists of overlapping quadrilateral cells, :math:`\mathcal{K}`.
We use harmonic polynomials, i.e polynomials that satisfy Laplace's equation, to approximate the solution :math:`\phi_h`.
In 2D these polynomials can be generated by taking the real and imaginary part of :math:`(x + iy)^n` for increasing :math:`n`.
The first ten such polynomials are:

.. math::
    \begin{array}{llll}
      f_1(x,y) =& 1                      & f_6(x,y)=& x^3 - 3 x y^2 \\
      f_2(x,y) =& x                      & f_7(x,y)=& 3 x^2 y - y^3\\
      f_3(x,y) =& y                      & f_8(x,y)=& x^4 - 6 x^2 y^2 + y^4  \\
      f_4(x,y) =& x^2 - y^2 \qquad\qquad & f_9(x,y)=& x^3y - x y^3 \\
      f_5(x,y) =& xy                     & f_{10}(x,y)=& x^5 - 10 x^3 y^2 + 5 x y^4
     \end{array}
